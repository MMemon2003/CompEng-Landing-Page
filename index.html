2nd Year - Greenhouse Monitoring System
Built a greenhouse system that monitors soil temperature (DS18B20), humidity (DHT22), and light (LDR), with all data shown on an LCD. Designed and soldered a custom PCB using Eagle.
Housed the system in a custom box featuring a rotary encoder to adjust temperature thresholds. The LDR was partially covered to limit light exposure. The setup enables interactive monitoring and control of environmental conditions.
3rd Year Mini Project - IOT Network Microcontroller System
Created an IoT-enabled system using ESP8266 and Tasmota firmware for I2C, SPI, and serial communication.
Integrated with Microsoft Azure for cloud connectivity and used Node-RED, Grafana, and InfluxDB for real-time monitoring.
GP Virtual Monitoring System (3rd Year Final Project)
Monitored temperature, blood pressure, and heart rate using DS18B20 and SEN0203 sensors. Used Tasmota and MQTT over Azure, with MySQL/XAMPP for local data storage.
Built a website (HTML/CSS/JS/PHP) for real-time health data access.
